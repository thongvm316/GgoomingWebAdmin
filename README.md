# Project: Ggooming Admin Web

### 1. Setup ReactJS App via Create React App

> Link: https://create-react-app.dev/docs/getting-started/

### 2. Install dependencies

```
npm i
```

### 3. Start for development

```
npm run dev
```

### 3. Build for production

```
npm run build
```
